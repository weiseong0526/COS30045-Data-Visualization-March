Q1 Storyboard (6 Panels)

Issue

The Australian market offers a narrow range of TV screen technologies, with very few OLED models.

Demonstrate Issue

LCD (LED): 167 models; LCD: 115 models; OLED: 10 models.

Ideas for Overcoming

Encourage wider OLED stocking; Introduce more mid-range OLED options; Improve buyer awareness of technology trade-offs.

Pilot Program

Goals: Increase OLED model availability; Run promotional campaigns; Add in-store technology guidance.

Before and After Data

Before: OLED 10 models, low buyer visibility. After: target +35 OLED models and stronger comparison choices.

Recommendation

Expand support for low-availability technologies; Keep mainstream options while improving premium-tech access.

This chart shows the size bands most available in stores. It supports practical decisions such as matching room size and budget to common model availability.

Q2 Storyboard (6 Panels)

Issue

Buyers are unclear which screen-size bands are most available in Australia.

Demonstrate Issue

65-inch: 494 models; 55-inch: 466 models; 75-inch: 335 models.

Ideas for Overcoming

Guide users to common size bands first; Match room size with top-availability categories; Improve size-based filtering support.

Pilot Program

Goals: Start filtering at 55/65/75 inch bands; Reduce search time; Improve shortlist quality.

Before and After Data

Before: random size selection and low comparability. After: focus on dominant sizes with highest model counts.

Recommendation

Use top size bands as default entry point; Compare energy and brand only within selected size band.

Q3 Storyboard (6 Panels)

Issue

Buyers struggle to identify brands with broad model variety.

Demonstrate Issue

Samsung: 1197 models; Kogan: 769 models; LG: 679 models.

Ideas for Overcoming

Prioritise top-variety brands; Reduce low-coverage brand noise; Build shortlist using model-count evidence.

Pilot Program

Goals: Use brand-ranking-first workflow; Improve shortlist speed; Increase relevant choices.

Before and After Data

Before: equal brand scanning. After: top-ranked brands produce larger and faster shortlists.

Recommendation

For Q3, shortlist top model-count brands, then filter by technology, size, and efficiency.

Q4 Storyboard (6 Panels)

Issue

Users cannot easily identify the lowest-power screen technology.

Demonstrate Issue

LCD: 89.28W; LCD (LED): 109.93W; Plasma: 116.17W; OLED: 128.76W.

Ideas for Overcoming

Make power metric visible early; Prioritise lower-mean-power technologies; Explain efficiency-cost impact.

Pilot Program

Goals: Apply power-first technology filter; Compare annual cost impact; Validate final user preference fit.

Before and After Data

Before: technology selected without power evidence. After: LCD baseline clearly identified as lowest-power option.

Recommendation

For Q4, prefer lower-mean-power technology where it still meets viewing preferences.

Q5 Storyboard (6 Panels)

Issue

The impact of larger screen size on energy use is not obvious to buyers.

Demonstrate Issue

<60cm: 81.26 kWh/year average; 80cm+: 441.15 kWh/year average.

Ideas for Overcoming

Show size-energy trend clearly; Flag high-consumption size ranges; Encourage balanced size decisions.

Pilot Program

Goals: Provide energy estimate by size band; Compare decisions before and after guidance.

Before and After Data

Before: size selected mainly by preference. After: users see clear energy difference between small and large sizes.

Recommendation

For Q5, balance preferred size with expected energy impact before final purchase choice.

Q6 Storyboard (6 Panels)

Issue

Buyers cannot easily judge whether star ratings improve or decline across screen sizes.

Demonstrate Issue

Average star <60cm: 6.72; Average star 80cm+: 7.27.

Ideas for Overcoming

Combine size and star in one decision step; Highlight high-star options per size band; Reduce low-efficiency selections.

Pilot Program

Goals: Add minimum-star filter by size; Measure shortlist quality and efficiency outcomes.

Before and After Data

Before: star rating ignored in size choice. After: size decisions include star-efficiency evidence.

Recommendation

For Q6, select higher-star models within the chosen size range to improve efficiency.
