# COS30045 - Data Visualisation
## Exercise 0.2 - TV Energy Data Story Website

This website presents findings from Australian TV market and energy datasets (`Q1.xlsx` to `Q6.xlsx` in `Exercise 3`).
Current story focus (Phase 1) is on **Q1 and Q2**.

## Data Story

### Audience
- Primary audience: Australian TV buyers and households planning a TV purchase.
- Secondary audience: tutor and classmates evaluating visual storytelling quality.
- Characteristics: non-technical users, short attention span, decision-focused.

### What This Audience Wants To Know
- Which screen technologies are commonly available in the market?
- Which TV screen sizes are most frequently available?
- What should be checked first before deeper comparison (power, stars, and brand)?

### Visual Story Guidelines
- Put practical decisions first: availability and common options.
- Present one question per expandable section.
- Pair each chart with a supporting data table for transparency.
- Use short context text so the reader knows why each visual matters.

### Storyboard (Q1 and Q2 First)
1. **Issue framing:** buyers are uncertain about realistic TV options currently available.
2. **Q1 view:** show technology frequencies to identify mainstream technology types.
3. **Q2 view:** show size frequencies to identify dominant market size bands.
4. **Decision step:** combine Q1 and Q2 to create a practical shortlist of likely products.
5. **Next stage:** extend with Q3-Q6 for brand breadth, power consumption, and star rating relationships.

## About the Data

### Data Source
- Course-provided files in `Exercise 3`:
  - `Q1.xlsx`, `Q2.xlsx`, `Q3.xlsx`, `Q4.xlsx`, `Q5.xlsx`, `Q6.xlsx`

### Data Processing
- Q1: grouped screen technology availability counts.
- Q2: counted available TV records by screen size.
- Q3: aggregated brand model counts.
- Q4: compared mean mode power by screen technology.
- Q5-Q6: retained pair data for scatter relationship analysis.

### Privacy
- Dataset is product-focused and not intended to expose personal identity.
- No direct personal identifiers are displayed in the website story.

### Accuracy and Limitations
- Results depend on source file quality and completeness.
- Product availability can change over time as listings update.
- Some categories may be underrepresented due to collection limits.

### Ethics
- Visual design avoids misleading scales and hidden transformations.
- Story text avoids overclaiming causation where only correlation is shown.
- Presentation aims to support informed consumer choices, not promote brands.
